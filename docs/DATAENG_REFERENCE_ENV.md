# Data Engineering Reference Environment

Validated Environment:
- Python 3.12.13
- macOS ARM64
- Conda: dataeng

Core Stack:
- Pandas 3.0.3
- Polars 1.41.2
- DuckDB 1.5.4
- PyArrow 24.0.0

Modern Tooling:
- SQLGlot
- Great Expectations
- Pydantic
- Structlog
- Rich
- Typer

Performance Tooling:
- Msgspec
- Orjson
- RapidFuzz
- DiskCache
- PyInstrument

Purpose:
Reference environment used to compare Python 3.12 production readiness
against Python 3.15 beta and release candidate environments.
